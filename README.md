# Cloud-Drop 🚀
Modern Drag & Drop File Sharing App

## ✨ Features
- Drag & drop file uploads  
- Real-time progress bars
- Image/PDF preview
- Fully responsive design
- Clean modern UI

## 🛠️ Tech Stack
- React 18 + Vite
- TailwindCSS 3
- React Dropzone
